# grim-essentials

The first-party Grimoire starter pack: one bundle that installs the three
grim knowledge skills together.

| Member | What it teaches |
|--------|-----------------|
| `grim-usage` | Driving the grim CLI end to end |
| `ai-config-authoring` | Vendor-neutral AI-config authoring craft |
| `grim-authoring` | Packaging artifacts for grim publishing |

Members ride the floating `:0` major tag and the bundle publishes
unpinned, so skill patch releases reach you through plain `grim update` —
no bundle re-release needed.

## Install

```sh
grim add ghcr.io/grimoire-rs/bundles/grim-essentials:0
```
